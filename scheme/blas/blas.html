Gambit-C BLAS interface

This is a thin wrapper for level 1, 2 and 3 BLAS linear algebra routines for the Gambit Scheme system. It provides access to all CBLAS functions using f32vector and f64vector types defined in srfi-4.

All functions can potentially modify its arguments, so use them with care. To copy the vectors before passing them to these functions is a good practice to preserve functional style.

Download scheme code: gambit-blas-1.0.tar.gz

Requirements:

CBLAS has to be installed and cblas.h has to be available.

Installation

You need to compile the cblas interface in order to use it: 
tar zxvf gambit-blas-1.0.tar.gz
cd blas
make

Usage

To use the interface, load it: 
(load "blas")
Then you can define vectors and have fun: 
(define X (f32vector 1. 2. 3.))
(define Y (f32vector 7. 7. 7.))
(define N (f32vector-length X))
(define incX 1)
(define incY 1)
(blas:sdot N X incX Y incY)
-> 42
(define alpha 2.)
(blas:sscal N alpha Y incY)  ;; Y mutates here
Y
-> #f32(14. 14. 14.)

Documentation

Here is a BLAS Quick Reference from netlib.org.

Meaning of prefixes

S: real (f32vector)
D: double (f64vector)
C: complex (f32vector)
Z: complex double (f64vector)

Matrix types

GE: GEneral      GB: General Band
SY: SYmmetric    SB: Symmetric Band    SP: Symmetric Packed
HE: HErmitian    HB: Hermitian Band    HP: Hermitian Packed
TR: TRiangular   HB: Triangular Band   HP: Triangular Packed

Constants defined

blas:RowMajor  -> ORDER
blas:ColMajor  -> ORDER
blas:NoTrans   -> TRANSPOSE
blas:Trans     -> TRANSPOSE
blas:ConjTrans -> TRANSPOSE
blas:Upper     -> UPLO
blas:Lower     -> UPLO
blas:NonUnit   -> DIAG
blas:Unit      -> DIAG
blas:Left      -> SIDE
blas:Right     -> SIDE
